## 简介<a name="section132119717356"></a>

CAMERA\_SCREENSAVER\_APP组件, 作为轻量系统带屏设备上的屏保应用参考实现，提供用户屏幕保护能力。它主要实现了以下功能：

1、支持播放预设图片能力。

2、支持点击屏幕退出能力。
